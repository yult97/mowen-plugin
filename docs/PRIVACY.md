# 隐私权政策 (Privacy Policy)

**生效日期：2024年1月1日**

本隐私权政策旨在说明"墨问笔记助手"（以下简称"本插件"）如何收集、使用和保护您的信息。

---

## 1. 我们收集的信息

### 1.1 用户主动提供的信息

**API Key**：为了连接"墨问笔记"服务，本插件需要您提供 API Key。该 Key 仅保存在您浏览器的本地存储（Local Storage）中，不会上传到除"墨问笔记"官方接口外的任何服务器。

### 1.2 自动生成的信息

**网页内容**：当您点击"剪藏"或"保存"按钮时，本插件会读取当前网页的正文HTML、图片和元数据（如标题、URL），并通过安全连接发送至"墨问笔记"服务器进行存储。

**错误日志**：为了优化体验，插件可能会在本地记录运行错误信息（如保存失败的原因），但这些日志不会包含您的个人身份信息，也不会自动上传。

---

## 2. 信息的用途

我们仅将收集到的信息用于以下单一目的：

**实现剪藏功能**：将您指定的网页内容经处理后同步到您的墨问笔记账户中。

---

## 3. 信息安全

- **数据直连**：本插件与"墨问笔记"官方服务器直接通信，中间不经过任何第三方服务器转发。
- **本地存储**：您的敏感配置（如 API Key）均存储在 Chrome 扩展的本地加密存储区中，仅限本插件访问。

---

## 4. 第三方服务

本插件的核心功能依赖于"墨问笔记"提供的 API 服务。关于墨问笔记如何处理存储在其服务器上的数据，请参阅墨问官方的隐私政策。

# Privacy Policy

**Effective Date: January 1, 2024**

This Privacy Policy explains how "Mowen Note Assistant" (the "Extension") collects, uses, and protects your information.

---

## 1. Information We Collect

### 1.1 Information You Provide

**API Key**: To connect with the "Mowen Notes" service, the Extension requires your API Key. This Key is stored locally in your browser (Local Storage) and is **NOT** uploaded to any server other than the official "Mowen Notes" API endpoints.

### 1.2 Automatically Generated Information

**Web Content**: When you click "Clip" or "Save", the Extension reads the current webpage's body HTML, images, and metadata (e.g., title, URL) and transmits them securely to the "Mowen Notes" server for storage.

---

## 2. How We Use Information

We use the collected information solely for:

**Web Clipping**: Processing and syncing your selected web content to your Mowen Notes account.

---

## 3. Data Security

- **Direct Connection**: The Extension communicates directly with "Mowen Notes" servers without any third-party intermediaries.
- **Local Storage**: Sensitive data (like your API Key) is stored in the Chrome Extension's secure local storage.
